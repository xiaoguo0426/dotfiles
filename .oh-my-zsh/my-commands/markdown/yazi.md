

### 导航

| 键绑定       | 可选键                 | 行动               |
|--------------|----------------------|------------------------|
| `k`     | `↑`             | 向上移动光标       |
| `j`     | `↓`             | 向下移动光标       |
| `h`     | `←`             | 向左移动光标       |
| `l`     | `→`             | 向右移动光标       |
| `g` => `g`     |              | 将光标移到顶部       |
| `G`     |              | 将光标移到底部       |

### 选择


### 文件操作
| 键绑定       |  行动               |
|--------------|------------------------|
| `o`     |  打开选中的文件       |
| `O`     |  以交互方式打开选定的文件       |
| `Enter`     |  打开选中的文件       |
| `Shift` + `Enter`     |  以交互方式打开选定的文件(一些终端还未支持)       |
| `Tab`     |  显示文件信息       |
| `y`     |  复制选定文件       |
| `x`     |  剪切选定文件       |
| `p`     |  粘贴已复制的文件       |
| `P`     |  粘贴复制的文件（如果目标位置已存在则覆盖）       |
| `Y` or `X`     |  取消yank状态       |
| `d`    |  将选定的文件删除到垃圾桶       |
| `D`    |  永久删除选定文件       |
| `a`    |  创建一个文件（目录以 / 结尾）       |
| `r`    |  重命名选定文件       |
| `.`    |  切换隐藏文件的可见性       |
| `;`    |  运行 shell 命令       |
| `:`    |  运行 shell 命令（阻塞直到完成）       |
| `-`    |  符号链接指向已复制文件的绝对路径       |
| `_`    |  符号链接指向已复制文件的相对路径       |
| `Ctrl` + `-`    |  硬链接已删除文件       |

### 复制路径
| 键绑定       |  行动               |
|--------------|------------------------|
| `c` => `c`     |  复制文件路径       |
| `c` => `d`     |  复制目录路径       |
| `c` => `f`     |  复制文件路径       |
| `c` => `n`     |  复制文件名（不带扩展名）       |

### 文件过滤
| 键绑定        |  行动               |
|--------------|------------------------|
| `f`          |  过滤文件       |

### 查找文件
| 键绑定        |  行动               |
|--------------|------------------------|
| `/`          |  查找下一个文件       |
| `?`          |  查找上一个文件       |
| `n`          |  前往下一个找到的文件       |
| `N`          |  前往上一个找到的文件       |

### 搜索文件
| 键绑定        |  行动               |
|--------------|------------------------|
| `s`          |  使用 `fd` 按名称搜索文件       |
| `S`          |  使用 `ripgrep` 按内容搜索文件       |
| `Ctrl` + `s`          |  取消正在进行的搜索       |

### 排序
| 键绑定        |  行动               |
|--------------|------------------------|
| `,` => `m`    |  按修改时间排序       |
| `,` => `M`    |  按修改时间排序(反向)       |
| `,` => `b`    |  按创建时间排序       |
| `,` => `B`    |  按创建时间排序(反向)       |
| `,` => `e`    |  按文件扩展名排序       |
| `,` => `E`    |  按文件扩展名排序(反向)       |
| `,` => `a`    |  按字母排序       |
| `,` => `A`    |  按字母排序(反向)       |
| `,` => `n`    |  按自然排序       |
| `,` => `N`    |  按自然排序(反向)       |
| `,` => `s`    |  按大小排序       |
| `,` => `S`    |  按大小排序(反向)       |
| `,` => `r`    |  随机排序       |
